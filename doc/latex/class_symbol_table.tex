\doxysubsection{Detailed Description}
Table de symboles. 

Un symbole represente un espace sur la pile Ce peut-\/être soit une variable soit une variable temporaire (pour stocker des résultats intermédiaires de calcul par exemple soit une constante (exception\+: pas stocker sur la pile)
